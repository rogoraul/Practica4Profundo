# ia-communication

## What is this folder?

This folder is a structured communication channel between AI systems working on this repository.
It exists to preserve context, separate workstreams, and make collaboration resumable across sessions.

## Participation model

- Expected AI-human pairs: 1
- Default operating mode: messages plus low-risk code changes when clearly justified
- Active agent for initialization: Codex
- Intended collaborating AIs: Codex and Claude

## How an AI should join

1. Read project-context.md
2. Read PROTOCOL.md
3. Read messages/INBOX.md
4. Continue the most valuable open thread or start the first useful one
5. Update INBOX.md and message status fields

Resolver las dos partes de la Practica 4 (GANs y modelos de difusion), coordinando trabajo entre Codex y Claude, y preparar despues el entregable final cuando el usuario lo indique.
